## 问题：如何处理交易 Pending / Confirmed / Failed 状态？

---

## 🎯 标准回答（直接背这个）

**第一步：说明状态流转**
> "交易状态主要有三种：Pending 表示交易已提交到链上等待打包，Confirmed 表示交易已被矿工打包进区块，Failed 表示交易执行失败。"

**第二步：说技术实现**
> "在实际开发中，我会用 ethers.js 的 `sendTransaction` 发送交易，然后用 `wait()` 方法等待交易确认。同时封装一个 React Hook 来管理状态，用 useState 维护 pending、confirmed、failed 三个状态，并通过 provider 监听确认数。"

**第三步：说用户体验**
> "在 UI 层面，pending 时显示 loading 动画和确认数，confirmed 后显示成功提示和区块浏览器链接，failed 时展示具体错误信息。还会把交易记录存到 localStorage，方便用户查看历史。"

**第四步：说高级处理（加分项）**
> "如果交易一直 pending，可以通过提高 gas price 发送相同 nonce 的交易来加速。对于重要交易，我会等待 12 个确认数才算真正成功。还会做错误重试和交易队列管理。"

**总结一句话**
> "核心就是用 ethers.js 的 wait 方法监听状态，用 Hook 管理状态，给用户清晰的反馈。"

## 💡 快速代码示例（面试官让写代码就写这个）

```typescript
function useTransaction() {
  const [status, setStatus] = useState('idle'); // idle/pending/confirmed/failed
  const [txHash, setTxHash] = useState('');
  const [error, setError] = useState('');

  const sendTx = async (signer, transaction) => {
    try {
      setStatus('pending');
      
      // 发送交易
      const tx = await signer.sendTransaction(transaction);
      setTxHash(tx.hash);
      
      // 等待确认
      const receipt = await tx.wait();
      
      // 判断成功或失败
      if (receipt.status === 1) {
        setStatus('confirmed');
      } else {
        setStatus('failed');
        setError('Transaction reverted');
      }
    } catch (err) {
      setStatus('failed');
      setError(err.message);
    }
  };

  return { status, txHash, error, sendTx };
}
```

**说明：** "这个 Hook 封装了完整的交易流程，发送时设置 pending，用 wait 等待确认，根据 receipt.status 判断成功失败。"

## 一、交易状态流转

```
发起交易 → Pending（待确认） → Confirmed（已确认） / Failed（失败）
```

### 状态说明
- **Pending**: 交易已提交到区块链网络，等待矿工打包
- **Confirmed**: 交易已被打包进区块，并达到安全确认数
- **Failed**: 交易执行失败（Gas 不足、合约 revert 等）

## 二、核心实现方案

### 2. 使用 React Hook 实现

```typescript
import { useState, useCallback, useEffect } from 'react';
import { ethers } from 'ethers';

function useTransaction(provider: ethers.providers.Provider) {
  const [txStatus, setTxStatus] = useState<TransactionStatus>(TransactionStatus.IDLE);
  const [txHash, setTxHash] = useState<string>('');
  const [error, setError] = useState<string>('');
  const [confirmations, setConfirmations] = useState<number>(0);

  // 发送交易
  const sendTransaction = useCallback(async (
    signer: ethers.Signer,
    transaction: ethers.providers.TransactionRequest
  ) => {
    try {
      setTxStatus(TransactionStatus.PENDING);
      setError('');
      
      // 发送交易
      const tx = await signer.sendTransaction(transaction);
      setTxHash(tx.hash);
      
      // 等待交易确认
      const receipt = await tx.wait();
      
      if (receipt.status === 1) {
        setTxStatus(TransactionStatus.CONFIRMED);
      } else {
        setTxStatus(TransactionStatus.FAILED);
        setError('Transaction reverted');
      }
      
      return receipt;
    } catch (err: any) {
      setTxStatus(TransactionStatus.FAILED);
      setError(err.message || 'Transaction failed');
      throw err;
    }
  }, []);

  // 监听确认数
  useEffect(() => {
    if (!txHash || !provider) return;

    let isCancelled = false;

    const watchConfirmations = async () => {
      try {
        const tx = await provider.getTransaction(txHash);
        if (!tx || isCancelled) return;

        provider.on(tx.hash, (receipt) => {
          if (!isCancelled) {
            setConfirmations(receipt.confirmations);
          }
        });
      } catch (err) {
        console.error('Error watching confirmations:', err);
      }
    };

    watchConfirmations();

    return () => {
      isCancelled = true;
      provider.removeAllListeners(txHash);
    };
  }, [txHash, provider]);

  return {
    txStatus,
    txHash,
    error,
    confirmations,
    sendTransaction
  };
}
```

### 3. 完整的交易管理 Hook

```typescript
import { useState, useCallback } from 'react';
import { ethers } from 'ethers';

interface TransactionState {
  hash: string;
  status: TransactionStatus;
  confirmations: number;
  error?: string;
  receipt?: ethers.providers.TransactionReceipt;
}

function useTransactionManager(provider: ethers.providers.Provider) {
  const [transactions, setTransactions] = useState<Map<string, TransactionState>>(new Map());

  // 添加交易
  const addTransaction = useCallback((hash: string) => {
    setTransactions(prev => new Map(prev).set(hash, {
      hash,
      status: TransactionStatus.PENDING,
      confirmations: 0
    }));
  }, []);

  // 更新交易状态
  const updateTransaction = useCallback((hash: string, updates: Partial<TransactionState>) => {
    setTransactions(prev => {
      const newMap = new Map(prev);
      const tx = newMap.get(hash);
      if (tx) {
        newMap.set(hash, { ...tx, ...updates });
      }
      return newMap;
    });
  }, []);

  // 监听交易
  const watchTransaction = useCallback(async (hash: string, requiredConfirmations = 1) => {
    try {
      addTransaction(hash);

      // 等待交易被打包
      const receipt = await provider.waitForTransaction(hash, requiredConfirmations);

      if (receipt.status === 1) {
        updateTransaction(hash, {
          status: TransactionStatus.CONFIRMED,
          confirmations: receipt.confirmations,
          receipt
        });
      } else {
        updateTransaction(hash, {
          status: TransactionStatus.FAILED,
          error: 'Transaction reverted',
          receipt
        });
      }

      return receipt;
    } catch (err: any) {
      updateTransaction(hash, {
        status: TransactionStatus.FAILED,
        error: err.message
      });
      throw err;
    }
  }, [provider, addTransaction, updateTransaction]);

  // 发送并监听交易
  const sendAndWatch = useCallback(async (
    signer: ethers.Signer,
    transaction: ethers.providers.TransactionRequest,
    requiredConfirmations = 1
  ) => {
    const tx = await signer.sendTransaction(transaction);
    await watchTransaction(tx.hash, requiredConfirmations);
    return tx;
  }, [watchTransaction]);

  return {
    transactions,
    addTransaction,
    updateTransaction,
    watchTransaction,
    sendAndWatch
  };
}
```

## 三、UI 组件实现

```typescript
import React from 'react';

interface TransactionStatusProps {
  status: TransactionStatus;
  hash?: string;
  confirmations?: number;
  error?: string;
}

const TransactionStatus: React.FC<TransactionStatusProps> = ({
  status,
  hash,
  confirmations = 0,
  error
}) => {
  const getStatusColor = () => {
    switch (status) {
      case TransactionStatus.PENDING:
        return 'text-yellow-500';
      case TransactionStatus.CONFIRMED:
        return 'text-green-500';
      case TransactionStatus.FAILED:
        return 'text-red-500';
      default:
        return 'text-gray-500';
    }
  };

  const getStatusText = () => {
    switch (status) {
      case TransactionStatus.PENDING:
        return `处理中... (${confirmations} 确认)`;
      case TransactionStatus.CONFIRMED:
        return `已确认 (${confirmations} 确认)`;
      case TransactionStatus.FAILED:
        return '交易失败';
      default:
        return '未开始';
    }
  };

  return (
    <div className="transaction-status">
      <div className={`status ${getStatusColor()}`}>
        {getStatusText()}
      </div>
      
      {hash && (
        <a 
          href={`https://etherscan.io/tx/${hash}`}
          target="_blank"
          rel="noopener noreferrer"
          className="text-blue-500 hover:underline"
        >
          查看交易详情
        </a>
      )}
      
      {error && (
        <div className="text-red-500 mt-2">
          错误: {error}
        </div>
      )}
      
      {status === TransactionStatus.PENDING && (
        <div className="mt-2">
          <div className="animate-spin rounded-full h-6 w-6 border-b-2 border-blue-500" />
        </div>
      )}
    </div>
  );
};
```

## 四、实际使用示例

```typescript
function TransferComponent() {
  const { provider, signer } = useWeb3();
  const { txStatus, txHash, error, confirmations, sendTransaction } = useTransaction(provider);
  const [recipient, setRecipient] = useState('');
  const [amount, setAmount] = useState('');

  const handleTransfer = async () => {
    if (!signer) return;

    try {
      await sendTransaction(signer, {
        to: recipient,
        value: ethers.utils.parseEther(amount)
      });
      
      alert('转账成功！');
    } catch (err) {
      console.error('转账失败:', err);
    }
  };

  return (
    <div>
      <input
        type="text"
        placeholder="接收地址"
        value={recipient}
        onChange={(e) => setRecipient(e.target.value)}
      />
      <input
        type="text"
        placeholder="金额 (ETH)"
        value={amount}
        onChange={(e) => setAmount(e.target.value)}
      />
      <button 
        onClick={handleTransfer}
        disabled={txStatus === TransactionStatus.PENDING}
      >
        {txStatus === TransactionStatus.PENDING ? '处理中...' : '发送'}
      </button>

      <TransactionStatus
        status={txStatus}
        hash={txHash}
        confirmations={confirmations}
        error={error}
      />
    </div>
  );
}
```

    this.queue.push(tx);

### 2. 交易重试机制

```typescript
async function sendTransactionWithRetry(
  signer: ethers.Signer,
  transaction: ethers.providers.TransactionRequest,
  maxRetries = 3
) {
  let lastError;
  
  for (let i = 0; i < maxRetries; i++) {
    try {
      const tx = await signer.sendTransaction(transaction);
      const receipt = await tx.wait();
      return receipt;
    } catch (err: any) {
      lastError = err;
      
      // 如果是 nonce 太低，不重试
      if (err.code === 'NONCE_EXPIRED') {
        throw err;
      }
      
      // 如果是 Gas 不足，增加 Gas
      if (err.code === 'INSUFFICIENT_FUNDS') {
        transaction.gasLimit = transaction.gasLimit?.mul(120).div(100);
      }
      
      // 等待后重试
      await new Promise(resolve => setTimeout(resolve, 2000 * (i + 1)));
    }
  }
  
  throw lastError;
}
```

### 3. 交易加速（Replace/Cancel）

```typescript
async function speedUpTransaction(
  signer: ethers.Signer,
  originalTx: ethers.providers.TransactionResponse,
  gasPriceMultiplier = 1.2
) {
  const newGasPrice = originalTx.gasPrice?.mul(Math.floor(gasPriceMultiplier * 100)).div(100);
  
  const newTx = await signer.sendTransaction({
    to: originalTx.to,
    value: originalTx.value,
    data: originalTx.data,
    nonce: originalTx.nonce,
    gasPrice: newGasPrice,
    gasLimit: originalTx.gasLimit
  });
  
  return newTx;
}

async function cancelTransaction(
  signer: ethers.Signer,
  originalTx: ethers.providers.TransactionResponse
) {
  const newGasPrice = originalTx.gasPrice?.mul(120).div(100);
  
  // 发送一个 0 ETH 到自己的交易，使用相同的 nonce
  const address = await signer.getAddress();
  const cancelTx = await signer.sendTransaction({
    to: address,
    value: 0,
    nonce: originalTx.nonce,
    gasPrice: newGasPrice
  });
  
  return cancelTx;
}
```

### 4. 本地存储持久化

```typescript
function usePersistedTransactions() {
  const [transactions, setTransactions] = useState<Transaction[]>(() => {
    const saved = localStorage.getItem('transactions');
    return saved ? JSON.parse(saved) : [];
  });

  useEffect(() => {
    localStorage.setItem('transactions', JSON.stringify(transactions));
  }, [transactions]);

  const addTransaction = (tx: Transaction) => {
    setTransactions(prev => [...prev, tx]);
  };

  const updateTransaction = (hash: string, updates: Partial<Transaction>) => {
    setTransactions(prev =>
      prev.map(tx => tx.hash === hash ? { ...tx, ...updates } : tx)
    );
  };

  return { transactions, addTransaction, updateTransaction };
}
```

## 八、常见面试追问

**Q: 如何判断交易真正成功？**
A: 检查 receipt.status === 1，并等待足够的确认数（通常 12 个区块）

**Q: 如果交易一直 Pending 怎么办？**
A: 可以通过提高 Gas Price 发送相同 nonce 的交易来加速或取消

**Q: 如何处理网络切换时的交易？**
A: 监听 chainId 变化，清理或标记不同网络的交易

**Q: 大量交易如何管理？**
A: 使用队列、分页、过滤器，只监听最近的活跃交易
